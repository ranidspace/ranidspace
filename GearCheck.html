Splatoon 3 Gear Completion Checker V1.1.0

This is a tool by Ranidspace designed to display all Splatoon gear you own sorted by how many stars they have

The geardata file can be obtained from the "/s3gear" command from the Spyke Discord bot (that link is to add the bot to your server), or from s3s, by using the "--getseed" argument. These files are usually for Lean's Gear Seed Checker

Custom fonts extracted from Splatoon 3 by North-West-Wind
Images and much of the data used in the backend by LeanYoshi

Everything is processed locally on your computer and nothing is uploaded, read the source code for this whole website here
